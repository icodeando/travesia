## Travesia

An application built to showcase random photos that I took in 2016 while I was exploring New York City. 

**Link to project:** https://travesia.netlify.app/

### How it's made:

**Tech used:** HTML, CSS, SASS, and JavaScript.

- Design by HTML5. 

- Deployed to Netlify.
